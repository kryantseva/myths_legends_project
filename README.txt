[Старт / Неавторизованный пользователь]
    ↓
    [Главная страница (Карта) - /]
        - Отображение интерактивной карты
        - Маркеры мест с базовой информацией (название, описание, категории, изображение, рейтинг, заметки)
        - Кнопка "Моя геолокация":
            → Запрос геолокации браузера
            → Отображение маркера пользователя
            → Центрирование карты на пользователе
            → Загрузка ближайших мест (с отображением расстояния в Popup)
        - Функционал избранных мест: возможность добавлять места в избранное.
        - Расширенный поиск и фильтрация: фильтры по категориям, статусу, поиск по тексту (доступны на карте и/или в списке мест).
        - Система рейтинга и отзывов: возможность пользователей ставить оценки и оставлять развернутые отзывы (через Заметки).
    ↓
    [Навигация (через Header - компонент Layout)]
        - Ссылки для всех пользователей:
            - "Карта" → [Главная страница (/)]
            - "Список Мест" → [Список Мест (/places)]
            - "Войти" → [Страница Входа (/login)]
            - "Регистрация" → [Страница Регистрации (/register)]
        - Ссылки для авторизованных пользователей:
            - "Профиль (username)" → [Страница Профиля (/profile)]
            - "Выйти" → Выход из системы
        - Ссылки для Модераторов/Администраторов:
            - "Модерация" → [Страница Модерации (/moderation)]

[Страница Входа (/login)]
    - Форма ввода логина/пароля
    → Успешный вход: Переход на [Главная страница (/) или Профиль (/profile)]

[Страница Регистрации (/register)]
    - Форма регистрации (имя пользователя, email, пароль)
    → Успешная регистрация: Переход на [Страница Входа (/login)]

[Страница Профиля (/profile)]
    - Отображение базовых данных пользователя (username, email)
    - Доступно только авторизованным пользователям

[Список Мест (/places)]
    - Отображение общего списка мест.
    - Включает функционал просмотра мест.
    - Добавление/редактирование мест пользователями через фронтенд:
        - "Добавить место" → [Страница Добавления/Редактирования Места (/places/add)]
        - "Редактировать место" → [Страница Добавления/Редактирования Места (/places/:id/edit)]
    - Применение расширенного поиска и фильтрации.

[Страницы Добавления/Редактирования Места (/places/add, /places/:id/edit)]
    - Форма для ввода информации о месте (название, описание, координаты, категории, изображение)
    - Редактирование существующих мест.
    - Принадлежность к пользователю (owner).

[Страницы Добавления/Редактирования Заметки (/places/:placeId/add-note, /notes/:noteId/edit)]
    - Функционал создания, просмотра, редактирования и удаления заметок пользователей к местам.
    - Включает возможность добавления оценки и изображения к заметке.
    - Возможность оставлять комментарии к заметкам.

[Страница Модерации (/moderation)]
    - Отдельный фронтенд-интерфейс для модераторов/администраторов.
    - Возможность одобрения/отклонения мест, заметок и комментариев.
